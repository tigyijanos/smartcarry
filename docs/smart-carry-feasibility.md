# Smart Carry Feasibility Notes

## Why This Should Be A Separate Mod

Carry capacity is a character-stat and balance concern.
It should be separated from hauling orchestration and routing logic.

## What Makes A Good Carry Model

- It is based on gameplay-relevant inputs.
- It produces one effective result other systems can consume.
- It is easy to explain and test.
- It does not depend purely on cosmetic appearance.

## What Would Be Weak Design

- Pure random per-settler carry values
- Permanent bonuses based only on cosmetic sex/body presentation
- Different carry-capacity rules in different subsystems
